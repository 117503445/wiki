# 基于 Ventoy 的移动硬盘多系统

## 背景

购买了 RC20 2t，并装入了移动硬盘盒作为移动硬盘使用。然后我选择了 Ventoy，支持在此硬盘上启动 Windows10 / 11 安装镜像, 微 PE, Windows To Go, ArchLinux With KDE。具体安装教程见参考链接。
