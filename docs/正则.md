## 例子

### 匹配单行注释

Java Go C ...

`//[\s\S]*?\n`

Python

`#[\s\S]*?\n`

### 匹配以 ** 开头的单词

利用零宽断言

比如 `(?<=No module named ')\w+`

从

ModuleNotFoundError: No module named 'bs4'

提取出 bs4
